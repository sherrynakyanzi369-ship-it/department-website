# Bugema University - Department of Computing & Informatics

A modern, highly responsive, and user-centric departmental mini-website designed to showcase the excellence, academic programs, and faculty of the School of Science & Technology at Bugema University.

## 🌟 Key Features

- **Premium Academic Design System**: Built utilizing a strict custom Navy Blue (`#003366`) and White color palette to maintain professional institutional precision.
- **Glassmorphic UI & Dynamic Layouts**: Features modern frosted-glass cards, soft drop shadows, and responsive UI foundations.
- **Micro-Interactions**: Includes a custom JavaScript typewriter hero animation, CSS stagger fade-ins on page load, and seamless smooth card hover effects.
- **Fully Responsive**: Optimized seamlessly for desktop, tablet, and mobile viewing featuring a custom mobile navigation system.

## 📂 Project Structure

The project uses a clean multi-page architecture:

- `index.html`: The main landing page with an engaging hero visual and centered quick-link grid block.
- `about.html`: Details regarding the departmental philosophy, mission, and pathways to faculty profiles.
- `academic-programs.html`: A comprehensive, two-column split layout detailing Computing and Life Sciences degrees.
- `admissions.html`: Clear guidelines and calls-to-action for prospective students.
- `research.html`: Highlighting innovative research centers and ongoing local/global projects.
- `news.html`: Latest events, tech symposiums, and departmental announcements.
- `staff.html`: Faculty and administration profile directory (accessible via the About page CTA).
- `contact.html`: A clean, minimalist flat-design contact form layout featuring an embedded Bugema University Google Map.
- `alumni.html`: An elegant profile showcase of distinguished graduates and their real-world impact.
- `styles.css`: The massive central core stylesheet housing CSS variables, typography, and complex animation logic.
- `script.js`: Global functional scripts handling mobile hamburger menus and dynamic states.

## 🛠️ Technology Stack

- **HTML5**: Utilizing modern semantic web structure.
- **CSS3 (Vanilla)**: Built natively without bulky frameworks. Relies on Flexbox, CSS Grid, and scoped custom variables for maximum browser performance and raw design control.
- **JavaScript (Vanilla)**: Lightweight interactions, including dynamic typers and responsive view toggles.
- **FontAwesome 6**: Vector icons embedded reliably throughout the grid items and contact systems.

## 🚀 Getting Started

1. Clone or download the repository to your local machine.
2. Ensure all assets (`logo.webp`, `hero-campus.png`, and dummy profile images) are located directly in the root directory.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari) to launch the application. No build step or local host server is explicitly required since the project consists of pure static files!
